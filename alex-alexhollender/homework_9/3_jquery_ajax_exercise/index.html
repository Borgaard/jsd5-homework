Two different methods of displaying information from openweathermap.org on a page via API 'get' calls (with jQuery).

Method 1 works in response to buttons being pressed. I store the different city IDs in the data-id attribute of the buttons, and then use those in the API call.

Method 2 prints out the information after the page has loaded. This this uses a for each loop to iterate over an array of objects I setup with the city IDs and the city names.
